# **🎨 SanStudio Image Toolkit**

**"Your images never leave your device" 🔒**

SanStudio Image Toolkit is a production-ready, highly modular, browser-based image editor. Built with a stunning UI inspired by Android 16 and OriginOS 6, it leverages the HTML5 Canvas API to deliver high-performance image manipulation without ever uploading your files to a server.

## **✨ Core Features**

### **📥 Import Engine**

* **File Upload & Drag-and-Drop:** Seamlessly drop images into the workspace.  
* **Clipboard Support:** Paste images directly (Ctrl+V / Cmd+V).  
* **Camera Capture:** Direct mobile camera access for instant editing.

### **🛠️ Editing & Transformation**

* **High-Fidelity Crop:** Freeform or standard aspect ratios (1:1, 16:9, 4:3, etc.).  
* **Smart Resize:** Pixel-perfect resampling with aspect-ratio locking.  
* **Transformations:** Rotate to any degree, 90° quick-turns, horizontal/vertical flipping.  
* **Hardware Accelerated Pan & Zoom:** Smoothly navigate large images with mouse wheels or pinch-to-zoom gestures.

### **🖌️ Creative Overlays (Layering Engine)**

* **Freehand Draw:** MS Paint-style drawing with adjustable brush sizes and colors.  
* **Text Watermarks:** Add customized text with Google Fonts integration, shadows, and dynamic scaling.  
* **Shapes:** Stamp rectangles, circles, triangles, hearts, and stars.  
* **Emoji Stamper:** Instantly overlay popular emojis perfectly scaled to your image.  
* **Image Collages/Overlays:** Upload secondary images, scale, and stamp them over your main canvas.

### **💾 Export & History**

* **Multi-Format Export:** Save as PNG, JPG, or WEBP.  
* **Quality Slider:** Adjust compression levels for JPG/WEBP exports to save space.  
* **Clipboard Export:** Copy the finalized image straight to your clipboard.  
* **Robust Undo/Redo:** 10-step history stack to easily revert mistakes.  
* **Session Recovery:** Accidentally closed the tab? Your last edit is safely restored from localStorage.

### **💅 UI/UX Design System**

* **Glassmorphism:** Beautiful frosted glass toolbars and floating panels.  
* **Dark Mode:** System-aware automatic dark mode with a manual toggle.  
* **Fully Responsive:** Adapts perfectly from large desktop monitors down to mobile screens.  
* **Smooth Animations:** Spring-based transitions and fluid micro-interactions.

## **🚀 Getting Started**

Because SanStudio is completely backend-free, getting started takes literally two seconds.

1. **Clone the repository:**  
   git clone \[https://github.com/your-username/sanstudio-image-toolkit.git\](https://github.com/your-username/sanstudio-image-toolkit.git)

2. **Open the app:**  
   Simply double-click index.html to open it in your favorite modern browser (Chrome, Firefox, Safari, Edge).  
   *No npm install, no servers, no databases.*

## **🛠️ Technology Stack**

* **Core Engine:** Vanilla JavaScript (ES6+), HTML5 Canvas API, Web APIs (Clipboard, File, Touch).  
* **Styling:** Tailwind CSS (via CDN for instant prototyping).  
* **Icons:** Phosphor Icons.  
* **Typography:** Inter, Google Sans, Montserrat, Pacifico, Oswald.

## **🔒 Privacy Guarantee**

**Zero Server Architecture.**

Every pixel of image processing happens via the JavaScript engine running directly in your browser. At no point does this application make network requests to send your images, telemetry, or personal data to an external server. It works completely offline once loaded.
